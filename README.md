# Body-Mass-Index calculator using Machine Learning

## Data Description:
1. Gender	-Male and Female
2. Height	- Range between 140-199
3. Weight	- Range between 50-160
4. Index - 
  - 1-Too Weak
  - 2-Weak
  - 3-Good health
  - 4-Overweight 
  - 5-Obesity
